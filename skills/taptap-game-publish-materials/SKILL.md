# TapTap 发布物料技能

## Overview

根据最终规则生成或校验 TapTap 发布物料。仅使用用户提供的资料与素材，不臆测游戏内容。
规则请见 `rules.md`。

## 使用流程

1. 选择工作方式（必须先问）
   - **自动读取**：读取现有代码与素材，根据规则生成全部物料与文案
   - **问询式**：一步步询问信息与素材，再生成或校验
   - **用户自理**：只输出需要的素材清单与规则，用户自行准备

2. 收集输入
   - 项目文档、代码、素材文件、已有成品图/视频/文案
   - 若关键信息缺失（尺寸、格式、数量、时长、文件大小、是否开放下载/试玩、是否 PC/主机、是否编辑推荐等），先提问补齐
   - 严禁根据常识推断游戏内容或功能；仅引用已提供资料

3. 优先使用用户已有素材
   - 识别到已有素材时，**优先复用并格式化**：裁切/导出/压缩/改格式/重命名
   - 若需要新素材：先问用户用什么工具生成
     - **AI 生图**（使用当前可用工具）
     - **用户指定工具**
     - **仅给出要求**，用户自行生成

4. 输出目录（必须生成）
   - 无论素材来源如何，**都复制一份**到结构化目录
   - 若用户不想复制，仍需给出该目录结构与文件命名清单

5. 生成后校验
   - 在输出目录逐项检查尺寸、格式、大小、数量、时长、比例与内容限制
   - 不符合项必须明确提示：问题点 + 需要如何修正

## 校验要求（概述）

- 依据 `rules.md` 的最终规则逐项校验
- 对素材做“格式化/转换”时，不改变核心内容含义
- 发现冲突或不确定项，必须回问用户
