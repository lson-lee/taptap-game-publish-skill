# TapTap 发布物料最终规则（客观要求）

仅包含最终可执行规则。若你需要“为什么”，请直接问用户，不要自行补充。

## 1. 文案与文本

- 游戏名：不得添加无关词缀与特殊符号
- 简介：仅用于介绍游戏本身；不得填写联系方式、外链或兑换码
- 开发者的话：不得填写联系方式、外链或兑换码
- 更新日志：不得出现宣传、广告、联系方式或其他无关内容（建议用有序列表）

## 2. 游戏图标

- 尺寸：不低于 512 x 512px
- 格式：JPG / PNG
- 形状：必须为直角方形，不可自行切圆角
- 底色：不得使用白底/黑底/透明底
- 安全区域：显示时会按约 22% 弧度裁切圆角，重要内容需避开裁切区域
- 其他：避免透明区域；避免重要信息贴边

## 3. 游戏截图

- 数量：不少于 3 张
- 格式：JPG / PNG
- 单张大小：4MB 以内
- 方向与比例：
  - 横图：宽高比 8:3 到 8:5，分辨率不低于 1280 x 720px
  - 竖图：宽高比 3:8 到 5:8，分辨率不低于 720 x 1280px
  - 所有图片比例需与第一张保持一致
- 内容要求：
  - 正式上线开放下载或开放试玩版时必须提供包含核心玩法的实机截图
  - 含实机画面的图片中，实机面积必须超过 50%
  - 概念类、广告宣传类图片最多 2 张，且实机截图需排在最前列
  - 不得仅包含商业内容（如充值、商城、抽卡）
- 其他：避免透明区域；避免清晰度过低；避免图片体积过大影响加载速度

## 4. 游戏实机录屏

- 必填条件：正式上线开放下载或开放试玩版时必须提供实机视频
- 格式：.mp4 或 .mov
- 编码：H.264 / HEVC (H.265)
- 大小：5GB 以内
- 时长：不少于 15 秒，最长不超过 30 分钟（建议控制在 2 分钟内）
- 比例与分辨率：
  - 横版视频比例 16:7 到 16:11，分辨率不低于 1280 x 720px
  - 竖版视频比例 7:16 到 11:16，分辨率不低于 720 x 1280px
- 内容要求：
  - 充分演示核心玩法；前 5 秒内展示核心玩法，且核心玩法展示至少持续 10 秒
  - 请勿展示纯商业化内容（如折扣充值）
- 禁止与限制：
  - 不得使用额外人声配音与字幕
  - 不得使用画中画
  - 不得露出手机等设备 UI
  - 不得拍摄游玩设备，仅使用游戏本身录屏画面
  - 不得含其他平台 LOGO 水印（如抖音/爱剪辑/快手/微博等）
  - 片尾不得含营销诱导性文案
  - 前 30 秒内避免在全屏画面中仅展示游戏/厂商/引擎 LOGO
  - 录屏内容占比需 ≥ 80%，不得额外使用人物立绘
- 画质：请提供高清视频

## 5. 宣传片

- 说明：可额外上传，仅支持横屏
- 格式：.mp4 或 .mov
- 编码：H.264 / HEVC (H.265)
- 大小：5GB 以内
- 比例与分辨率：16:9，分辨率不低于 1280 x 720px
- 内容：展示核心玩法为主
- 注意：视频前 3 秒避免仅有 logo/黑屏等无实机画面

## 6. 宣传图（16:9）

- 用途：
  1) 首页推荐、今日游戏等卡片展示页面
  2) 横版实机录屏、宣传片视频封面
  3) 未上传实机录屏时，作为详情页首张图集图片
- 尺寸：不低于 1920 x 1080px
- 格式：JPG / PNG
- 大小：4MB 以内
- 文本：必须包含游戏标题；不得出现游戏标题以外的宣传性文字
- 视觉：不得出现游戏图标；需高清精美；不得直接使用游戏截图或简单拼贴
- 形状与显示：直角方形；上下各预留 108px 渐变区域；展示时可能按 4:3 比例裁切
- 布局提示：将游戏 LOGO / 主要角色放在 4:3 裁切安全区域内
- 其他：不可自行裁剪；避免透明区域；避免清晰度过低；避免翻转/倒置；避免游戏名过大

## 7. 宣传图（1:1）

- 适用：作为竖版实机录屏视频封面时使用
- 尺寸：1440 x 1440px
- 格式：JPG / PNG
- 大小：1MB 内
- 形状：直角方形
- 显示：上下各预留 108px 渐变区域
- 内容一致性：除位置/大小适配外，主体内容需与 16:9 宣传图一致

## 8. 首页推荐语

- 要求：客观描述游戏亮点，避免“最/第一”等极限词或商业宣传语
- 字数：不超过 25 个字符

## 9. 首页推荐素材选择

- 可选项：
  - 使用宣传图 (16:9)（不使用任何视频素材）
  - 使用游戏实机录屏视频
  - 使用宣传片（首页推荐自动播放）

## 10. PC/主机封面与游戏库素材（如涉及 PC/主机）

### 10.1 游戏 LOGO
- 尺寸：满足 1280px 宽或 720px 高其一，可自定义宽高比
- 格式：png / jpg
- 要求：LOGO 元素尽量贴合实际尺寸，避免四周大边距/空隙；避免清晰度低或像素模糊
- 说明：LOGO 内非设计元素空间使用透明处理；避免出现 LOGO 本体以外的视觉设计元素
- 建议：使用识别度高、与背景对比度大的版本

### 10.2 横版游戏封面图
- 尺寸：460 x 215px
- 格式：png / jpg
- 大小：3MB 内
- 内容：带有清晰游戏 LOGO；清晰展现游戏名与宣传内容
- 注意：避免 LOGO 过小；避免低清/模糊；避免透明裁切/圆角；避免过于简洁导致识别性低
- 说明：该图作为识别名片，一般无需随版本频繁更换

### 10.3 竖版游戏封面图
- 尺寸：600 x 900px
- 格式：png / jpg
- 大小：3MB 内
- 内容：带有清晰游戏 LOGO；保证 LOGO 清晰且对比度高，尽量使用大尺寸 LOGO 排版
- 注意：避免 LOGO 过小；避免低清/模糊；避免透明裁切/圆角；避免过于简洁导致识别性低
- 建议：背景素材使用具有代表性的视觉元素

### 10.4 游戏库背景壁纸
- 尺寸：1920 x 1080px
- 格式：png / jpg
- 大小：3MB 内
- 内容限制：避免出现游戏 LOGO、宣传标题等任何文字内容
- 画质：高清且精美
- 布局提示：公告与活动 UI 可能覆盖左下角（最大宽度 400px）
- 布局提示：LOGO 自动布局在右上角，LOGO 下方建议使用干净背景
- 布局提示：主体元素建议放在画面中心偏右安全范围内

## 11. 首页编辑推荐位素材（仅编辑推荐游戏）

- 尺寸：1920 x 1280px（3:2）
- 格式：png / jpg
- 大小：5MB 内
- 文件提供：必须提供 png/jpg；建议提供 psd 源文件
- 提示：建议提交素材时提供 2–3 套图片，便于一次性通过审核
- 规范：图像清晰；避免拉伸失真；画面有明确视觉焦点
- 色彩：主色明确，避免颜色过多
- 文本：避免出现除游戏名以外的文字；游戏名面积需小于图片总面积的 25%；如需添加游戏名，以小标题形式放在画面
- 注意：不可自行裁剪；避免透明区域；避免清晰度过低
- 其他：避免视点不突出；避免色彩过多导致杂乱；避免低劣素材、明显 AI 作图或拼接痕迹（素材质量会影响是否能进入推荐位）
- 处理提示：若图片不符合素材要求，将不会加入编辑推荐位进行分发

## 12. 测试服/体验服（如上传）

- 需提供：安装包与使用说明、测试账号与密码、开关服时间

## 13. 其他表单字段提示

- 玩家交流群链接：仅输入链接地址，勿夹杂中文或其他非链接内容，否则链接可能失效
- 兼容性与系统要求、游戏官网链接、玩家群信息、是否提供 AI 合成服务等为表单字段，按实际填写
